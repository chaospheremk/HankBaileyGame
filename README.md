# The Zoomies 🐱

A cozy, charming endless runner starring **Hank** (a big friendly orange chonky boy) and **Bailey** (a skittish, weird black cat who does the sideways Halloween crab walk and stares at walls for no reason).

*3am. No reason. Full speed.*

## Play

Open `index.html` in a modern browser, or visit the [GitHub Pages site](https://chaospheremk.github.io/HankBaileyGame/).

## Controls

| Input | Action |
|---|---|
| Space / Up Arrow | Jump |
| S / Down Arrow | Duck |
| Tab / X | Swap active cat |
| Touch tap | Jump |
| Touch swipe down | Duck |
| Touch swipe left | Swap cats |

## About

Built as a single `index.html` — no build step, no dependencies, no external image or audio assets. All art is drawn programmatically on HTML5 Canvas and all sound is synthesized via the Web Audio API.

See [docs/PLAN.md](docs/PLAN.md) for the full design specification.
